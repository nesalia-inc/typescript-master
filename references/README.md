# Reference Guide

Detailed guidance for TypeScript topics.

## Topics

| Topic | File | Description |
|-------|------|-------------|
| Advanced Types | [advanced-types.md](./advanced-types.md) | Generics, conditional types, mapped types, template literals |
| Type Guards | [type-guards.md](./type-guards.md) | Type narrowing, discriminated unions, assertion functions |
| Utility Types | [utility-types.md](./utility-types.md) | Partial, Pick, Omit, Record, custom utilities |
| Configuration | [configuration.md](./configuration.md) | tsconfig options, strict mode, project references |
| Patterns | [patterns.md](./patterns.md) | Builder, factory, type-safe APIs |
| tRPC | [trpc.md](./trpc.md) | End-to-end type safety client ↔ server |

## Usage

Load the relevant reference file based on the task at hand:

- **Generics or complex types** → advanced-types.md
- **Type narrowing or state machines** → type-guards.md
- **tsconfig or build config** → configuration.md
- **tRPC integration** → trpc.md
